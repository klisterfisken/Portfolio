# Portfolio

School assignment as part of the frontend curriculum:  
Design a portfolio website that at least contains the following:

- Portrait picture
- Button for downloading CV/resume
- Some kind of navbar
- Links to social media / github
- About me section
- Knowledge / techstack section
- Portfolio section with projects
- Testimonials section
- Contact form
- Footer

I've used HTML, CSS and JavaScript as follows:

### HTML:  
The website is an responsive Single Page Application based out of index.html and I've aimed for a semantic structure to the best of my abilities.

### CSS:  
Not using any framework, I've styled the website using variables for colours, and using nesting css for media querys and responsive behaviors. For layout I've mostly used flexbox, but you will find grid for the quote carousel and the about me section on larger screens.

### JavaScript:  
I have used JavaScript as little as possible, mostly to push the limits of what CSS alone can achieve. I have used two different EventListeners: The header scroll animation is one of them and calling weather API on page load is the other.
